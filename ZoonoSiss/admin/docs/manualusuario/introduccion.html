1.1 Introduccion

El presente manual de usuario tiene como fin mostrar las diferentes funciones de la plataforma Web “INTERSHIP”, y así mismo, servir como ayuda para los diferentes clientes (Estudiante, Jefe de carreras, Tutor y Administrativo ) para que puedan tener un mayor acercamiento a la plataforma web y aprendan a usarlo de una manera correcta.

La página web tiene como funciones:

  • Manejar sesión usuario web: permite crear una cuenta con datos personales (nombre, apellido. Correo. Contraseña, ) y también permitirá cerrar sesión web e iniciar sesión (correo y contraseña).

  • Gestionar Informe Semanal: Permitir a los usuarios web ya sean usuarios, tutor,jefe de carrea, administrador o usuarios personal a llevar un control de las pasantias que se realizan, a ver la fecha de estos pasantes y a crear las pasantias e iniciarlos.

  • Ver los requisitos que necesita: permite a los usuarios estudiantes a obtener información sobre los requisitos que necesita para iniciar una pasantia.

  • Agregar datos: Dar la opción de agregar una pasantia , de crear un nueva pasantia para un estudiante, agregar tipo de pasantía y eliminar pasantía.

1.2 Objetivo de este manual

Se pretende mostrar de una manera clara y concisa el funcionamiento del Sistema de seguimiento de pasantias "INTERSHIP" en la fundación infocal.

  • Clasificar usuarios, mediante uso de privilegios en la gestión de usuarios autorizados.

  • Ingresar a la aplicación mediante un Nick y una contraseña

  • Validar los datos mediante requisitos planteadas por la institución.

  • Mostrar el Estado de los diferentes trámites en curso.

  • Optimizar las búsquedas de documentos relacionados a los trámites mediante números de folio relacionados en la aplicación.

  • Registrar el estado de los tramites en curso.

  • Generar reportes por fecha y tipo.

1.3 Dirigido a...

Este manual va dirigido a:

  • Estudiante.

  • Administrador.

  • Personal administrativo.

1.4 Lo que debo conocer

Los conocimientos mínimos que deben tener las personas que operarán las páginas y deberán utilizareste manual son:

  • Conocimientos básicos acerca de Programas Utilitarios.

  • Conocimiento básico de Internet.

  • Conocimiento básico de Windows
