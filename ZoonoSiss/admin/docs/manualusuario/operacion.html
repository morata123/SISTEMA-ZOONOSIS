Este capítulo proporciona la operacion del Sistema mostrando cómo utilizar el Sistema de Control de Pasantia para vizualizar, crear/modificar y hacer todo lo que puede hacer el sistema.

Este capítulo asume que el Sistema está ya iniciado en su máquina con su usuario. Si esto no es cierto, contacte a su administrador del Sistema.

Este capítulo describe todo el proceso de manejar el sistema con distintos tipos de usuarios.
